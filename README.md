# 💻 Full Stack Portfolio - Cleyton Barbosa

Repositório oficial do meu portfólio pessoal. Um ambiente desenvolvido para demonstrar minha evolução técnica, stack tecnológica e a transição estratégica de uma carreira de 8 anos de disciplina e liderança para a Engenharia de Software.

## 🧐 Sobre o Projeto

Este portfólio não é apenas uma vitrine visual, mas um laboratório de fundamentos web. Ele foi projetado para centralizar meus projetos mais relevantes, focando em:

- **Efeitos Visuais de Baixo Nível:** Implementação de animações e sistemas de partículas utilizando Canvas API e JavaScript Vanilla.
- **Arquitetura de Contato:** Sistema de modal customizado para gestão de leads/contatos profissionais.
- **Copywriting Estratégico:** Foco em converter visitas em oportunidades, destacando competências em **Backend (Node.js/PHP)** e **Mobile (React Native)**.

## 🛠️ Tecnologias & Conceitos

- **UI/UX:** HTML5 Semântico e CSS3 Avançado (Flexbox/Grid/Animations).
- **Interatividade:** JavaScript (ES6+) para manipulação de DOM e lógica de digitação (Typing Effect).
- **Performance:** Otimização de renderização e scripts assíncronos para um carregamento fluido.
- **Icons & Fonts:** Font Awesome e Google Fonts Integration.

## 🚀 Projetos em Destaque no Portfólio

Dentro do site, você encontrará detalhes sobre:
1. **CG-Finance:** Gestão financeira completa em React Native.
2. **Açaí Delivery:** Backend de logística e pedidos em Node.js.
3. **Escala Pro:** Sistema de regras de negócio complexas em PHP.
4. **Auth Security:** API de segurança com JWT e RBAC.
