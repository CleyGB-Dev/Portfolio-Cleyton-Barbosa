# 💻 Meu Portfólio Pessoal - Cleyton Barbosa

Este é o repositório do meu portfólio pessoal, onde apresento as minhas competências, tecnologias com as quais trabalho e os projetos que desenvolvi ao longo da minha jornada como desenvolvedor.

🚀 **Link para Visualização:** (https://cleygb-dev.github.io/Portfolio-Cleyton-Barbosa/)

---

## 🧐 Sobre o Projeto
O objetivo deste portfólio é centralizar a minha presença online, servindo como um currículo interativo e uma amostra do meu trabalho. Ele foi construído focado em:
- **Performance:** Carregamento rápido e otimizado.
- **Responsividade:** Adaptado para dispositivos móveis, tablets e desktop.
- **UX/UI:** Interface limpa e intuitiva para uma melhor experiência do utilizador.

## 🛠️ Tecnologias Utilizadas
As principais ferramentas utilizadas no desenvolvimento deste projeto foram:

- **HTML5** & **CSS3** (Estrutura e Estilização)
- **JavaScript (ES6+)** (Interatividade)
- **Git & GitHub** (Controlo de Versão e Deploy)
